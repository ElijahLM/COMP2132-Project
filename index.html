Dice Game

About This Game:

  • Press the roll button to roll the dice and after 3 rolls the game will end
  • Press the reset button to reset the game
  • Two dice will be rolled and tallied into a result for that round
  • If either dice rolls a 1 the player will have a result of 0 for that round
  • If the two dice rolls the same number the result will be doubled
